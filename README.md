# T2-Week5

# Senja Coffee (Landing Page)

Website ini merupakan project sederhana berupa **Landing Page Coffee Shop** yang dibuat menggunakan **HTML dan CSS** untuk memenuhi tugas 2 mata kuliah Pemrograman Web.

---

## Deskripsi

Senja Coffee adalah website landing page yang menampilkan informasi tentang sebuah coffee shop dengan tampilan modern dan sederhana.

Website ini berisi beberapa bagian utama seperti:

- Navbar (menu navigasi)
- Hero section (tampilan utama)
- About (tentang café)
- Menu (daftar produk)
- Features (keunggulan)
- Footer (informasi tambahan)

---

## Teknologi yang Digunakan

* HTML5
* CSS3

---

## Fitur Website

* Navbar sederhana dengan navigasi
* Hero section dengan background image dan overlay
* About section untuk deskripsi café
* Menu section menggunakan **CSS Grid**
* Features section menggunakan **Flexbox**
* Hover effect pada:

  * Navigation link
  * Button
  * Card menu
  * Feature box
* Desain clean dan minimalis

---

## Konsep yang Diterapkan

### 1. External CSS

Menggunakan file `style.css` untuk memisahkan struktur (HTML) dan desain (CSS).

### 2. Box Model

Menggunakan properti:

* margin
* padding
* border-box (box-sizing)

untuk mengatur layout dan jarak antar elemen.

### 3. Flexbox

Digunakan pada:

* Navbar
* Feature section

### 4. CSS Grid

Digunakan pada:

* Menu section (`menu-container`)

### 5. Hover Effect

Digunakan untuk memberikan interaksi pada user, seperti:

* perubahan warna pada navbar
* efek pada button
* animasi card saat hover
